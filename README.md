# RoadSense Logger

**RoadSense Logger** adalah aplikasi Android yang dirancang untuk mendukung **pengumpulan data perjalanan dan kondisi jalan** berbasis perangkat mobile, sebagai alat bantu awal dalam kegiatan **observasi, monitoring, dan analisis kondisi prasarana jalan**.

Aplikasi ini memanfaatkan sensor bawaan smartphone untuk merekam data selama perjalanan, sehingga dapat digunakan sebagai **pendukung pengambilan keputusan** pada tahap survei awal, studi lapangan, maupun evaluasi kondisi jalan.
